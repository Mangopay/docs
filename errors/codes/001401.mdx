---
title: '001401'  
---


                    | **Error message**     | Transaction has already been successfully refunded                   |
                    |-----------------------|--------------------------------------------|
                    | **Short description** | The transaction has already been successfully refunded. |
                    | **Categories**        | <div class="pill">Refund</div>                         |
                  

#### Causes  

This error occurs when attempting to make a Refund on a transaction that has already been refunded.

#### Testing  

You can trigger this error in the Sandbox environment using the Check Instant Payout eligibility endpoint. You just need to define the `BankWireRef` parameter to `IP1070`.  
```json Request example
{
    "AuthorId":"142036728",
    "DebitedFunds":{
        "Currency":"EUR",
        "Amount":1260
    },
    "Fees":{
        "Currency":"EUR",
        "Amount":12
    },
    "BankAccountId":"151986981",
    "DebitedWalletId":"145389978",
    "BankWireRef":"IP1070",
    "PayoutModeRequested":"INSTANT_PAYMENT"
}
```  
```json API Response
{
    "InstantPayout": {
        "IsReachable": false,
        "UnreachableReason": {
            "Code": "130010",
            "Message": "Generic operation error"
        }
    }
}
```

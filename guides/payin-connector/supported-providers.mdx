title: Supported providers

This values are used for the ExternalProviderName value of Mangopay's [settlement file format](//guides/payin-connector/mirakl#specifications-of-the-mangopay-format-settlement-file).

During the establishment of your contract with Mangopay, any new providers are added to the list.

## About

BLIK is a popular payment method available on mobile banking applications in Poland. It allows users to pay online, at physical terminals, and between individuals, as well as withdraw cash from ATMs.

## How it works  

<Steps>
    <Step>
        On your app or website, the user selects BLIK as the payment method during checkout.
    </Step>
    <Step>
        The user opens their banking app where they find a six-digit code generated.
    </Step>
    <Step>
        The user copies the code and returns to your app or website.
    </Step>
    <Step>
        The user enters the code to validate the payment.
    </Step>
</Steps>

## Outcome

The transaction is complete when the pay-in status changed from `CREATED` to `SUCCEEDED` or `FAILED`, indicating the outcome.  

You should also set up <a href="">hook notifications</a> fo<a href="">r the relev</a>ant event types:  
- PAYIN_NORMAL_SUCCEEDED  
- PAYIN_NORMAL_FAILED  

## Constraints

<Note>  
**Note - Minimum amount in Production** 
  
In Production, the minimum accepted amount for BLIK is 0.01 PLN (`1`).  
</Note>

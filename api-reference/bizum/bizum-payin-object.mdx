### Description

The Bizum PayIn object allows your platform to process Bizum payments.

There are two possible flows, both using the same endpoint with different payloads: 
- Phone – If the `Phone` parameter is sent, then the Bizum user receives a push notification to authenticate the payment.
- Redirect – If the `Phone` parameter is not present, the `ReturnURL` is required and a `RedirectURL` is returned on which you need to redirect the user to enter their phone number.

You can use this endpoint in three capture scenarios requiring three difference request bodies.
- **Full simultaneous capture** – The full intent is captured and the `ExternalData` is the same as the Intent (same provider references). 
  - Required: 
    - Call with no request body.
- **Full separate capture** – The `ExternalData` references are/or date are different from the Intent (for example, a capture following preauthorization). 
  - Required: 
    - `ExternalData.ExternalProcessingDate`
    - `ExternalData.ExternalProviderReference`
  - Optional:
    - `ExternalProviderPaymentMethod`
- **Partial capture** – The `ExternalData` references are/or date are different from the Intent and the Capture is for part of the full Intent amount. 
  - Required: 
    - `Amount`
    - `ExternalData.ExternalProcessingDate`
    - `ExternalData.ExternalProviderReference`
    - `LintItems.Id`
    - `LintItems.Amount`
  - Optional:
    - `ExternalProviderPaymentMethod`

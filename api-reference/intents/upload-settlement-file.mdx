---
title: Upload a settlement file
api: POST /v2.01/{ClientId}/payins/intents/settlements 
description: Send a Mangopay-format settlement file to Mangopay for a single currency
---

Use this endpoint to send Mangopay CSV settlement files in Mangopay’s defined template. 

You need to send one file per currency, as soon as it becomes available.

<Warning>
**Caution – Use settlement file example as template**

You must follow the [specifications provided here](/guides/payin-connector/mirakl#specifications-of-the-mangopay-format-settlement-file).

Use this [example CSV file](https://raw.githubusercontent.com/Mangopay/docs/refs/heads/main/guides/payin-connector/Example_Mangopay_Settlement_File_Mirakl.csv) (right click and ***Save as...***), which contains the correct column and row labels and example data for 4 transactions. Remove the example data lines before generating your own file. 

There are no requirements for the **file name**: you can name your file how you like.
</Warning>

### Body parameters

<ParamField body="file" type="csv">
The file to upload. The file must be in CSV format and conform to Mangopay's [format described here](/guides/payin-connector/mirakl#specifications-of-the-mangopay-format-settlement-file).
</ParamField>

### Responses

<AccordionGroup>
<Accordion title="200 - OK" defaultOpen>

This endpoint returns a JSON object containing details about the settlement (SettlementId, declared amount, settled amount, status, etc.)

</Accordion>
</AccordionGroup>

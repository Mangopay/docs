You can use this endpoint to cancel all or part of an Intent.

<Tabs>
<Tab title="Full cancellation">
The full `Amount` of the intent is canceled (all line items). The `ExternalData` references are/or date must be provided as cancellation was necessarily a different action taken via the third-party PSP.
</Tab>
Required: 
- `ExternalData.ExternalProcessingDate`
- `ExternalData.ExternalProviderReference`
<Tab title="Â¨Partial cancellation">
Part of the original Intent `Amount` is canceled, meaning one or more line items but not all. The `ExternalData` references are/or date are different from the Intent. The cancellation `Amount` must equal the sum of its line items.
  - Required: 
    - `Amount`
    - `ExternalData.ExternalProcessingDate`
    - `ExternalData.ExternalProviderReference`
    - `LintItems.Id`
    - `LintItems.Amount`
</Tab>
</Tabs>

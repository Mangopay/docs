### Description
The UBO Declaration object is a container to provide information regarding the ultimate <BeneficialOwner content="beneficial owners"/> of a legal entity. It is mandatory for legal users with `LegalPersonType` of `BUSINESS` or `PARTNERSHIP`.

See the <a href="/guides/glossary#beneficial-owner">glossary</a> for details on the criteria defining beneficial owners.

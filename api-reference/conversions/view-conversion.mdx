<Note>  
**Note â€“ QuoteId and Fees returned for all conversions** 
  
All conversions returned by this endpoint contain the `QuoteId` and `Fees` parameters.   
For instant conversions, the `QuoteId` is `null`. For quoted conversions, the `Fees` amount is zero.  
</Note>

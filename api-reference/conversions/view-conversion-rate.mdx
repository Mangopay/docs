<Note>  
**Note – Rate is not guaranteed** 
  
The rate returned by the API is indicative of the rate offered by Mangopay, but it is not a guaranteed rate. When you Create an Instant Conversion, the actual rate used during the transaction is displayed in the response.   
</Note>

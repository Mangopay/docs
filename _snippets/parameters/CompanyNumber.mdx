<ParamField body="CompanyNumber" type="string">
  **Required if :** `UserCategory` is `OWNER` and `LegalPersonType` is `BUSINESS`

  The registration number of the entity, assigned by the relevant national authority. For information on the expected format for a specific country, see the [Requirements for legal users](https://mangopay.com/docs/concepts/users/verification/requirements-legal). To validate the format of a number before submitting documents for verification, use [Validate the format of User data](https://mangopay.com/docs/endpoints/user-data-format#validate-user-data-format).

  This value is null if the `UserCategory` is `PAYER`.
</ParamField>

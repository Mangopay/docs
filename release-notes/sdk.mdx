## Stay up to date

GitHub provides notification features that you can use to receive notifications when Mangopay publishes a new SDK release for your language.

### Watch the repo

<Steps>
<Step title="Click watch on the repo homepage">
On the homepage of the repository you're using, click ***Watch*** then ***Custom***. For example, on the <a href="https://github.com/Mangopay/mangopay2-net-sdk/releases" target="_blank">.Net SDK</a> homepage:
<img src="/images/screenshot_github_watch_1.png" alt="screenshot of the GitHub repository for the Mangopay .Net SDK with the Watch button clicked"/>
</Step>
<Step title="Select releases">
In the dialog, select ***Releases*** and click ***Apply***. 
<img src="/images/screenshot_github_watch_2.png" alt="screenshot of the subsequent dialog showing the Custom option" />
</Step>
</Steps>

Configure your email notifications in your account's <a href="https://github.com/settings/notifications" target="_blank">profile settings</a>, and read more about GtiHub's notification features on <a href="https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/setting-up-notifications/about-notifications" target="_blank">their documentation</a>.

### Send notifications to Slack

<Steps>
<Step title="Add the Slack app">
Add the <a href="https://slack.github.com/" target="_blank">GitHub Slack app</a> to your workspace.
</Step>
<Step title="Subscribe to releases">
You can receive notifications to a channel or your personal GitHub app conversion.

To see all commands, type:
```
/github help
```
To subscribe to releases on Mangopay's .Net SDK, for example, type:
```
/github subscribe mangopay/mangopay2-net-sdk releases
```
</Step>
</Steps>

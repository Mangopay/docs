Changelog of new features and improvements in the <a href="https://hub.mangopay.com/" target="_blank">Mangopay Dashboard</a>. 

---

## October 1, 2025

<div class="pill pill-green">Added</div>
- [Role-based access control (RBAC)](/dashboard/team) and new team management section deployed to second batch of platforms:
  - **Action required for platforms using custom roles:** Assign new roles to the relevant users to ensure a seamless transition to RBAC (head to <a href="https://hub.mangopay.com/" target="_blank">Dashboard</a> -> ***Company profile*** -> ***Team*** for details)

<div class="pill pill-blue">Improved</div>
- Production and Sandbox environments more clearly distinguished to increase security, and switcher component enhanced for usability
- Styling for refined Mangopay branding, including improvements to various UI components

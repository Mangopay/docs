Changelog of new features and improvements in the <a href="https://hub.mangopay.com/" target="_blank">Mangopay Dashboard</a>. 

## February 3, 2025

<div class="pill pill-green">Added</div> 
- Refund support for bank wire pay-ins and pay-ins to virtual IBANs (see [Refunds using payouts](/guides/payouts#refunds-using-payouts) for details)

---

## January 17, 2025

<div class="pill pill-green">Added</div>
- Refunds possible directly from the card transaction details page

<div class="pill pill-blue">Improved</div>
- **FX client rate and market rate** shown on conversion details
- **Links to each page** of the new Dashboard thanks to dedicated URLs
- **Card fingerprint data** shown on card details
- Various usability improvements to environment selector, search, bank accounts, transfers, etc.

---

## December 12, 2024 

<div class="pill pill-green">Added</div> 
- **Dedicated Help Center** for more comprehensive handling of Support tickets
- **Wire reference** field in bank wire pay-in details

<div class="pill pill-blue">Improved</div> 
- Updated **account details statement PDF** (replacing RIB)
- Event type naming consistency in webhooks
- Date of birth input field on user creation

---

## November 22, 2024 

<div class="pill pill-green">Added</div>
- Initiate a **direct debit pay-in via mandate**, and view direct debit pay-in details
- **Manage mandates** attached to a user's bank account, including mandate cancellation and PDF download
- Pay-in details for new payment methods: Swish, TWINT

<div class="pill pill-blue">Improved</div>
- Behavior when switching between Sandbox or Production environments, giving smoother experience
- T&Cs checkbox on user creation
- Various minor bug fixes

---

### SpotFX: Instant Conversions now available  

Platforms can now execute an immediate conversion of funds between two wallets of different currencies (both owned by the same user), thanks to the Create an Instant Conversion endpoint.   

The rate offered by Mangopay can be viewed in real-time via the View an indicative Conversion Rate endpoint.   

Mangopay’s SpotFX feature allows platforms to offer their users multi-currency wallet balances across GBP, EUR, and USD.   
<a href="#" class="btn Secondary normal">Learn more about FX</a>  

#### Endpoints  

The following endpoints have been added:   
-View an indicative Conversion Rate  
-Create an Instant Conversion  
-View an Instant Conversion  

#### Webhooks  

The following event types are available for hook notifications:  
-INSTANT_CONVERSION_IN_PROGRESS  
-INSTANT_CONVERSION_SUCCEEDED  
-INSTANT_CONVERSION_FAILED  

#### Errors  

The following functional errors have also been added:  
-004801: Conversion could not be debited from wallet  
-004802: Conversion failed during conversion operation

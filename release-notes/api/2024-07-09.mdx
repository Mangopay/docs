## Web Direct Debit PayIn endpoint marked as deprecated

The [Web Direct Debit PayIn](/api-reference/web-direct-debit-payins/create-web-direct-debit-payin) endpoint has been marked as deprecated and should no longer be used to request pay-ins.

The two payment methods available via this endpoint, Giropay and Sofort, are no longer available.

## Giropay PayIn endpoint marked as deprecated

The [Giropay PayIn](/api-reference/giropay/create-giropay-payin) endpoint has been marked as deprecated and should no longer be used to request pay-ins. Giropay is no longer available as of June 30, 2024, but refunds are still possible during 1 year.

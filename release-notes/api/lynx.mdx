<span class="date">2019-05-24</span>  
  
  
<div class="pill pill-green">Added</div>  
  

In order to help you through the changes induced by the fourth Anti-Money Laundering, we have added new filters on the GET /emoney/ endpoint allowing you to check the amount per month.  
<div class="pill pill-blue">Changed</div>
  

For platforms using mandates: we improved the way we handle the GoCardless events (mandates provider). We use to process the events as they come, we will now process them by creation date. This will vanish confusions you may have had.

<ResponseField name="Scheme" type="string">
**Allowed values:** One of the [supported schemes](/guides/payment-methods/banking/pay-by-bank/supported-banks) for the `Country`.

**Default value:** The [listed instant scheme](/guides/payment-methods/banking/pay-by-bank/supported-banks) for the `Country`. If `Country` is `DK`, there is no default and the `Scheme` must be specified.

The scheme to use to process the payment. Note that some banks may charge additional fees to the user for instant payment schemes.
</ResponseField>

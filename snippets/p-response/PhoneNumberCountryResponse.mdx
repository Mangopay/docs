<ResponseField name="PhoneNumber" type="string">
**Allowed values:** Two-letter country code (ISO 3166-1 alpha-2 format).

Required if the `PhoneNumber` is provided in local format.

The country code of the `PhoneNumber`, used to render the value in the E.164 standard.

**Caution:** If `UserCategory` is `OWNER`, modifying this value means the user will be required to re-enroll the new value in SCA via the `PendingUserAction.RedirectUrl`. For more details see the [SCA guides](/guides/users/sca/enrollment).
</ResponseField>

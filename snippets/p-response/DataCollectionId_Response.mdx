<ResponseField name="DataCollectionId" type="string">
The unique identifier of the Data Collection object containing the data sent in the request, which must be added to the PayPal pay-in. 

A new `DataCollectionId` is needed for each call to [POST Create a PayPal PayIn](/api-reference/paypal/create-paypal-payin), [POST Create a Recurring PayPal PayIn (CIT)](/api-reference/paypal/create-recurring-paypal-payin-cit), and [POST Create a Recurring PayPal PayIn (MIT)](/api-reference/paypal/create-recurring-paypal-payin-mit).
</ResponseField>

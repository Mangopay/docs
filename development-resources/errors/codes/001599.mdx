                    | **Error message**     | Token processing error                   |
                    |-----------------------|--------------------------------------------|
                    | **Short description** | The token for the card wasn't created. |
                    | **Categories**        | <div class="pill">Card tokenization</div><div class="pill">Card registration</div>                         |
                  

#### Causes  

This error occurs when the tokenization server does not receive any data during the tokenization HTTP call.  

It can be due to a timeout, or the call being blocked by an anti-virus, plugin, or extension.

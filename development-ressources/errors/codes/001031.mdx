 title: '001031'  
---  


                    | **Error message**     | User canceled the payment                   |
                    |-----------------------|--------------------------------------------|
                    | **Short description** | The pay-in failed because the user canceled the payment. |
                    | **Categories**        | <div class="label">Web pay-in</div>                         |
                  

#### Causes  

This error occurs when the user cancels the payment.  

#### What to do  

If the end user did not deliberately cancel the payment, they should retry.

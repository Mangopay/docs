 title: '001030'  
---  


                    | **Error message**     | User has not been redirected                   |
                    |-----------------------|--------------------------------------------|
                    | **Short description** | The pay-in failed because the user has not been redirected to the payment page. |
                    | **Categories**        | <div class="pill">Web pay-in</div>                         |
                  

#### Causes  

This error occurs when the end user does not access the payment page. Most often, this error is due to the fact that the end user deliberately did not access the payment page.  

#### What to do  

In such cases, if this is not due to a deliberate action, then the end user should retry.
